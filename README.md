# Autonomous UAV Navigation Using GNSS and AI

An advanced autonomous UAV navigation and control system that combines GNSS positioning, IMU sensor fusion, AI-based navigation, and real-time telemetry monitoring for intelligent unmanned aerial vehicle operations.

---

# Project Overview

This project simulates and demonstrates an intelligent UAV navigation framework capable of:

- Autonomous waypoint navigation
- GNSS-based positioning
- IMU sensor integration
- AI-assisted path planning
- Kalman Filter sensor fusion
- Obstacle-aware trajectory handling
- Real-time telemetry monitoring
- Simulation environment for UAV testing

The system is designed as a research-level project suitable for advanced embedded systems, aerospace, robotics, and AI applications.
